Simple Calculator

Simple Calculator is a calculator application coded with React Native.
The project was coded based off of a tutorial by Kyle W. Banks
(https://kylewbanks.com/blog/react-native-tutorial-part-2-designing-a-calculator).

Files:

SimpleCalculator.js - The main code which runs the app.
InputButton.js - The code for each Input Button.
Style.js - The StyleSheet.
